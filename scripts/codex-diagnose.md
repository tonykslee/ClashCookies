## Codex-diagnose task format

Issue:
<what is wrong in 1-3 lines>

Observed:
- <actual behavior>
- <logs/error>
- <when/where it happens>

Expected:
- <expected behavior>

Likely scope:
- <command/file/service>
- <recent related change if known>

Ask:
- identify root cause
- point to exact file/code path
- propose minimal fix
- do not implement yet unless explicitly asked

Output:
- root cause
- affected files
- minimal patch plan
- risk/regression notes
